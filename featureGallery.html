Features and Resources

Minecraft: Education Edition offers many features and resources - including worlds, kits, and special items that are inspired by a wide range of laerning subjects.

Area & Volume

This is a yearlong project to discover the magic of area and volume.

Coordinates

Learn the techniques of navigation using the Minecraft Coordinate System.

Subtraction

Develop extensive subtraction skills with help from this Minecraft world.

Chemistry

Use Minecraft's unique chemistry tools to enguage students in the classroom.

Biodiversity

Explore these resources for interactive lessons about ecosystems, conservation, etc.

Manufacturing

This collection of activities familiarises students in the concept of manufacturing.

Python 101

Python kit includes syntax, variables, conditional loops, functions, lists, and others.

Hour of Code 2021

Using your coding superpowers, save the future by solving mysterious mishaps in time.

Events (Coding)

Learn how to use Events in JavaScript and Python with MakeCode Coding Editor.
